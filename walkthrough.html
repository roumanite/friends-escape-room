Escape Room
  1. Go to the bathroom and pick up the purple box placed inside the bathtub.
  2. Go to the living room and lift the yellow pillow. Pick up the yellow notepad.
  3. Click the laptop. The yellow notepad hints that the laptop password is a translation of "Joey Tribbiani". Type in "Baby Kangaroo Tribbiani".
  4. A hint for the purple box's code will be displayed. The code is:
    1. Monica's apartment number in that episode - 20
    2. How many claps are there in the theme song? - 4
    3. Which day is Thursday? - 3
    4. Chandler's apartment number in that episode - 19
    5. How many steaks are there? - 5
    6. How many eggplants are there? - 1
    7. How many pages did Ross have to read? - 18
    The code is 2043195118.
  5. Pick up the jellyfish protection lotion once the box is opened.
